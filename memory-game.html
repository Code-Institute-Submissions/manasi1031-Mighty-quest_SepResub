Memory Game

Match pairs of food cards in this fun memory game.

How to play?

Start by clicking on any card to start timer.
Each card has a image of a fruit, meal or a vegetable.
Choose two facing down cards to turn them over.
If it is a match, the cards will be removed from the game.
Otherwise they will be turned over again.
The game ends when you match all the pairs.
